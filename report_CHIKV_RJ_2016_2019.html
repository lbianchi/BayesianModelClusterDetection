1 Lendo dados e carregando pacotes

Os números de notificações de chikungunya são referentes ao periodo de 01/01/2016 até 31/12/2019 para todos os municipios do estado do Rio de Janeiro. Os dados foram obtidos atráves da API do Infodengue. O número de casos foi obtido somando as semanas epidemicas de cada municipio e as 10 aps do Rio de Janeiro por ano. Para o calculo da incidência foi considerado como população sucetível os dados obtidos no DataSUS para os anos de 2015 a 2018, sendo esses utilizados para o calculo da incidência do ano consecutivo.

2 Objetivo

O objetivo desse documento é modelar a incidência de chikungunya para os 92 municipios do estado do Rio de Janeiro visando detectar a ocorrência de grupos (clusters espaciais) para cada ano.

RJ.map.sf <- rbind(RJ.map.sf_2016,RJ.map.sf_2017,RJ.map.sf_2018,RJ.map.sf_2019) %>% 
  mutate(ano = sort(rep(2016:2019,nrow(CODMUN.tbl))),
         IncidenciaCat = cut(mean, breaks = c(0, floor(as.vector(quantile(mean,probs = seq(0.1,0.9,0.1)))), Inf),
                             labels = paste0(c("  0",seq(10,90,10)),"% --| ",seq(10,100,10),"%"))
  )
#Getting descriptive stats of incidence
